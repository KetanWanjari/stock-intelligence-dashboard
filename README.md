## Project Overview

This project implements a **full-stack stock market intelligence platform** that:

- Generates realistic **NSE stock market data** (10 major Indian companies)
- Cleans, transforms, and stores data in a **SQLite database** via Pandas
- Exposes **REST APIs** using FastAPI with auto-generated Swagger docs
- Serves an interactive **Chart.js dashboard** with real-time filtering
- Includes a **Linear Regression ML model** for price prediction

In a production environment, the mock data generator (`data_generator.py`) would be replaced by a live feed using `yfinance`, `Alpha Vantage`, or the NSE bhavcopy CSV download.

## Setup & Installation

### Prerequisites

- Python 3.10 or higher
- pip

### Steps

```bash
# 1. Clone / unzip the project
cd stock_dashboard

# 2. (Optional but recommended) Create a virtual environment
python -m venv venv
source venv/bin/activate        # macOS / Linux
# OR
venv\Scripts\activate           # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. (Optional) Pre-generate the database manually
python data_generator.py
```

## Running the App

```bash
uvicorn main:app --reload --port 8000
```

Then open your browser at:

- **Dashboard:** http://localhost:8000
- **Swagger API Docs:** http://localhost:8000/docs
- **ReDoc API Docs:** http://localhost:8000/redoc

> The database is auto-created on first startup if `stocks.db` does not exist.

## API Endpoints

### Core Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/companies` | List all available companies |
| GET | `/data/{symbol}?days=30` | Last N days of OHLCV + metrics |
| GET | `/summary/{symbol}` | 52-week high/low/avg + key stats |

### Bonus Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/compare?symbol1=INFY&symbol2=TCS&days=90` | Compare two stocks (normalized) |
| GET | `/gainers-losers` | Top 5 gainers and losers for the latest date |
| GET | `/predict/{symbol}?horizon=7` | ML-based price prediction |

### Example Requests

```bash
# All companies
curl http://localhost:8000/companies

# 30-day data for Infosys
curl http://localhost:8000/data/INFY?days=30

# 52-week summary for TCS
curl http://localhost:8000/summary/TCS

# Compare INFY vs RELIANCE over 90 days
curl "http://localhost:8000/compare?symbol1=INFY&symbol2=RELIANCE&days=90"

# Predict HDFC Bank next 7 days
curl http://localhost:8000/predict/HDFCBANK?horizon=7
```

| ITC | ITC Ltd | FMCG |
| BAJFINANCE | Bajaj Finance Ltd | Finance |
| AXISBANK | Axis Bank Ltd | Banking |

### Required Metrics

| Metric | Formula |
|--------|---------|
| Daily Return | `(CLOSE - OPEN) / OPEN × 100` |
| 7-day Moving Average | Rolling mean of close (window=7) |
| 30-day Moving Average | Rolling mean of close (window=30) |
| 52-Week High | Max of `HIGH` over last 365 days |
| 52-Week Low | Min of `LOW` over last 365 days |

### Custom Metrics (Bonus)

| Metric | Description |
|--------|-------------|
| **Volatility Score** | 20-day rolling standard deviation of daily returns. Higher = more volatile stock. |
| **RSI (14-day)** | Relative Strength Index. >70 = overbought (sell signal), <30 = oversold (buy signal). |
| **Momentum (20-day)** | `(Close / Close_20_days_ago - 1) × 100`. Measures trend strength. |

## Dashboard Features

The single-page frontend (`/`) provides:

1. **Company List Sidebar** — click any stock to load its data
2. **Stock Panel:**
   - 7 key stat cards (close, 52W high/low, avg, RSI, volatility, return)
   - Filterable by range: 30D / 90D / 6M / 1Y
   - Three charts: Closing Price + MAs, Daily Returns (bar), RSI
3. **Compare Tab:**
   - Pick any two stocks, choose a time window
   - Normalized price chart (both start at 100)
   - Side-by-side stats + return correlation coefficient
4. **Gainers / Losers Tab:**
   - Top 5 gainers and top 5 losers for the most recent trading day
5. **Predict Tab:**
   - Select any stock + horizon (7/14/30 days)
   - Calls the `/predict` ML API and renders predicted prices

## ML Prediction Feature

**Model:** `sklearn.LinearRegression`

**Features used:**
- Close price (lag 1, 2, 3, 5 days)
- 7-day and 30-day moving averages
- Volatility score
- RSI (14-day)
- 20-day momentum

**Training:** On all historical data for the selected symbol.

**Prediction:** Auto-regressive — each predicted close becomes an input for the next day's prediction.

**Limitation:** Linear regression is intentionally simple. For production, LSTM or XGBoost would be used. Results are for educational/demo purposes only.

## Design Decisions & Insights

### Why SQLite?
Lightweight, zero-config, and perfect for a self-contained demo. Switching to PostgreSQL requires only changing the connection string and using `psycopg2`.

### Why Geometric Brownian Motion for mock data?
GBM is the standard model for stock price simulation — it captures both the random walk nature and the positive drift of equity markets. The parameters (μ=0.03% daily drift, σ=1.5% daily vol) are realistic for large-cap Indian stocks.

### Interesting Observations from the Mock Data
- **Correlation between IT stocks** (INFY, TCS, WIPRO) is high (>0.7) — they tend to move together due to USD earnings exposure.
- **BAJFINANCE** has the highest volatility score among all companies — consistent with the real-world behavior of high-growth NBFC stocks.
- **RSI signals** can identify overbought/oversold periods useful for swing trading decisions.
